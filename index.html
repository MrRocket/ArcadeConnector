Arcade Connector is a Windows-based launcher for CSUME: the Client/Server Universal Machine Emulator.

It brings multiplayer arcade games to life with modern features like IRC chat, ready-up autojoin, profile images, and game broadcasting.

Features

  • Auto-join via Ready-Up
  • Integrated IRC chat with status-aware icons
  • Clickable join:// links for fast connection
  • Profile pictures and sounds
  • UDP-hosted server detection (works without IRC)
