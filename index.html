Arcade Connector is a Windows-based launcher for C.S.U.M.E: a Client/Server Universal Machine Emulator. Based on M.A.M.E but with client server networking in mind. Arcade Connector brings multiplayer arcade games to life with modern features like IRC chat, ready-up autojoin, profile images, and game broadcasting.

  

    • Auto-join via Ready-Up
    • Integrated IRC chat with status-aware icons
    • Clickable join:// links for fast connection
    • Profile pictures and sounds
    • UDP-hosted server detection (works without IRC)
    • Hosted Servers Browser (works without IRC)

    • Windows 7/10/11 with .NET Framework 4.5+
    • Minimum hardware: Dual-core CPU, 1GB RAM, Direct3D-capable GPU
    • Internet or LAN connection
      (Offline gameplay still works, but hosting and joining require a network)
    • CSUME (included)
    • Ports 5029 (CSUME), 45291 (broadcast)
