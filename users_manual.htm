User Manual  

 

 

 

 

1. Getting Started

Arcade Connector is a Windows launcher designed for hosting and joining multiplayer arcade games using CSUME.
Follow these steps to prepare your environment:

Requirements

  • Windows 7 / 10 / 11
  • .NET Framework 4.5+
  • CSUME executable (included)
  • Your own ROM files in csume/roms/
  • A stable internet connection for hosting/joining games.

 

When you first run Arcade Connector you be presented with the IRC login screen tab. It will try to detect if you are connected. IRC is optional and isn't required, but it will make things much easier to chat and find hosted games.

  1. Create a user name or login annymously. If no name is created, a random user name will be given. Starting with User and a random number.
  2. Click Connect and login to IRC.

2. Hosting a Game

  1. Launch Arcade Connector.
  2. Choose a ROM file from Browse ROM.
  3. Configure hosting options (engine, parameters, etc.).
  4. Click Host Game.
  5. A join:// link will be posted to IRC so others can join.
  6. Once done, close CSUME to return to the launcher.

Note: If you're not using IRC, your server will show up in the Hosted Servers browser.

3. Joining a Game

  • Look for join:// links in the chat window.
  • Click the link to connect to the hosted.
  • If you notice that a game is prepared to be hosted, Right click your user name and select "Ready up" to auto join a game. ~ This is actually the best way to join. It keeps the game in sync and will take less time to join.

Note: If you're not using IRC, detected hosted servers will show up in the Hosted Servers browser. Double Click to Connect.

4. User Status Icons

  • Online — User is connected and active.
  • AFK — User is idle or has typed /afk.
  • In-game — User is currently in a game.
  • Ready — User has toggled Ready Up to auto-join games.
  • Hosting — User is hosting a game.

When set to host a game by ticking the check box "Host Game", the user icon automatically changes notifying others that you're prepared to host a game:

Once everyone is ready to Join, Click the Host Game button to start your game server.

Note: If Host Game is not checked, only a single player game will be started, by clicking Launch Game.

5. IRC Commands

  • /afk — Set your status to AFK.
  • /back — Clear AFK status.
  • /sound [filename] — Play a custom sound (e.g., /sound cheer).
  • /image [filename] — Post an inline image from ChatImages (if enabled).

6. Chat Features

  • Clickable Links
    URLs starting with http://, https://, or join:// become clickable.
  • Posting Images
    If a URL ends with .png, .jpg, or .bmp, it will attempt to show the image inline.
  • Sound Commands
    Type /sound [soundfile]

Here's a list of sound that are currently supported:

  • /sound cheers
  • /sound beep
  • /sound powerup
  • /sound beback
  • /sound message
  • /sound doomdoor
  • Image Commands
    Type /beer, /soulsphere, etc., to insert predefined images from Images/.
